anual de Usuario – Siruma Resort
1. Nombre del proyecto

Siruma Resort – Sistema de Gestión del Hotel

2. Introducción

Este programa permite gestionar de forma básica las operaciones del Resort usado en la lúdica.
El sistema funciona por consola y permite:

Hacer el alistamiento inicial (habitaciones, turismo y alimentación).

Registrar los clientes que van llegando.

Manejar turismo, alimentación y mascotas.

Calcular ventas, costos y ganancia.

Ingresar a un módulo administrador con usuario y contraseña.

Consultar detalles de los clientes registrados.

El objetivo es facilitar el manejo del día en el resort y llevar el control de forma organizada.

3. Requisitos del sistema

Python 3

Consola o terminal

4. Opciones del sistema

A continuación se explica cada opción del menú:

4.1. Planeación del alistamiento

Permite ingresar cuántas habitaciones y servicios hay disponibles para el día.
Se registran los cupos para:

Habitaciones sencillas

Habitaciones dobles

Habitaciones familiares

Turismo

Alimentación

Si un servicio se agota, el cliente pasa a extraordinario automáticamente.

4.2. Registrar un cliente

El usuario escoge el tipo de cliente:

Individual

Pareja

Familia

Luego se registran los datos de cada persona del grupo:

Nombre

Apellido

Documento

Después se selecciona si desean:

Turismo

Alimentación

Mascotas

El sistema calcula automáticamente:

Si el servicio es general o extraordinario

Cuánto paga el cliente

Cuánto cuesta al resort

4.3. Módulo administrador

Para entrar se debe usar:

Usuario: admin

Contraseña: 1234

Dentro se pueden ver los reportes:

Total de clientes (personas)

Total por tipo de habitación

Total de mascotas

Disponibilidad de habitaciones

Disponibilidad de turismo y alimentación

Ventas totales

Costos totales

Ganancia

Detalles completos de clientes

Nombres

Documentos

Servicios tomados

Si fueron generales o extraordinarios

Costos

4.4. Salir del sistema

Finaliza la ejecución del programa.
